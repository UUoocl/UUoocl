Make a Family Tree Fan Chart

This thing is a new Family Tree Chart for the Photo Family Tree template. A fan chart is a creative display of a person’s direct ancestors. FamilySearch.org has a fan chart view, which this thing is based on. If you want to create a fan chart but don’t want to enter your family information on FamilySearch, then you might like this.

This chart shows the ancestors of a single person in your family tree. The chart looks like this,

The template can be cusomized. Here is an example in with a water color style.

To create the chart you’ll need Inkscape and an extension Next Generator. Both are free!

Follow along to make your own.

There are 2 parts to the Fan Chart Template.

  1. a Google Sheets Template where the family tree data is saved
  2. an Inkscape Scabalbe Vector Graphic (SVG) file.

1. Get a Copy of the Family Tree Fan Chart Google Sheets template

Copy the Google sheet Photo Famly Tree with Fan Chart template to your Google Drive. Use the “People” tab to enter you family tree information. Watch this video to learn how to get started.
Create a Photo Family Tree with Google Sheets and yEd

Add Fan Chart to Existing Spreadsheet

If you already have a family tree template you can add the “FanChart” tab.

  • Right click on the “FanChart” Tab.

  • Select “Copy To”

  • Then select the spreadsheet to add the sheet to.

2. Fill In the Fan Chart Sheet

Use Column A to select a Person. Each row in the Fan Chart Sheet will create a fan chart diagram.

3. Download the Fan Chart info

In the Family Tree Template, select the “Fan Chart” sheet. Click

  • File ➡️ Download ➡️Comma Separated Values (.csv)

Now that the Fan chart data is downloaded from the Google Sheet, it’s time to work on creating the fan Chart diagram in Inkscape.

4. Create the Fan Chart Diagram in Inkscape

  • Download Inkscape

    Inkscape is a free drawing application. https://inkscape.org/ Install and Open Inkscape

  • Download the Inkscape Extension NextGenerator

    The NextGenerator Extension will match the data from the CSV file to the Fan Chart template. NextGenerator Extension

The extension will need to be saved to a folder named ‘User extensions’. Here’s how to find the ‘User extensions’ folder.

  • In Inkscape, click Edit ➡️Preferences

  • Select ‘System’, then click the ‘open’ button next to ‘User extensions’

The 2 NextGenerator files that will be saved in the User extensions folder are

  • next_gen.inx
  • next_gen.py
  • Download the next_gen.inx and next_gen.py files to the “User extensions” folder.
    • Close and reopen Inkscape.

Download the Family Tree Fan Chart Template

Family Tree Fan Chart Template

Open the Fan Chart Template in Inkscape

  • File ➡️Open

  • Open the NextGenerator Extension

    • Extensions ➡️Export➡️NextGenerator
      • Choose the location of the CSV file
      • Enter %VAR_Name% as the file name
      • Choose a folder to save the Fan Charts
        • Create the folder, if it doesn’t exist.
      • Press the ‘Apply’ button

The template uses the free Google Font “Courier Prime”. This font can be optional installed. Courier Prime Font
